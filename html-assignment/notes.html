NOTES

List of 5 HTML tags I have learned with its corresponding descriptions:

  • <head> - This tag contains information about the webpage, such as the title, encoding, links to files and other settings.
  • <title> - This tag defines the title of the webpage. This appears on the browser tab and is also used by search engines to identify the page.
  • <p> - The paragraph tag is used to display blocks of text. It automatically adds space before and after the text to separate it from the other elements.
  • <ul> - This tag is called as the unordered list. This tag creates a bulleted list and is used when the order of the items do not matter.
  • <li> - This list item tag defines individual items inside the <ul> or <ol> lists.
